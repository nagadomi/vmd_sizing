# Ubuntu port of VMD Sizing

## Installation (Ubuntu 20.04)

```
sudo apt-get install python3 python3-pip python3-wxgtk4.0 python3-numpy
pip3 install numpy-quaternion bezier
```

Run
```
python3 src/executor.py
```
